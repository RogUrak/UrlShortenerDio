# (DIO) Construindo Encurtador de URL

Desafio de construir uma API utilizando NodeJS e Typescript para gerar hashes Ãºnicos para cada entrada, e conectar as entradas com os hashes gerados.

## Rodando o projeto

- `npm install`
- `npm run build`
- `npm run dev`

### **Endpoints:**

- Encurtador de URL: POST /shorten
- Interpretador da URL encurtada: GET /:hash
- Status da API: GET /status
